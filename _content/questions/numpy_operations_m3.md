Given the following NumPy arrays:

```python
import numpy as np
a = np.array([1, 2, 3])
b = np.array([4, 5, 6])
```

Which of the following operations performs an element-wise addition of the arrays?

A) 
```python
result = np.add(a, b)
```

B) 
```python
result = np.dot(a, b)
```

C) 
```python
result = np.matmul(a, b)
```

D) 
```python
result = np.outer(a, b)
```

<!-- Answer: A) -->
