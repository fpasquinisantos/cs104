Given the following NumPy arrays:

```python
import numpy as np
x = np.array([10, 20, 30])
y = np.array([1, 2, 3])
```

What will be the result of the following operation?

```python
result = x + y
```

A) 
```python
array([11, 22, 33])
```

B) 
```python
array([10, 20, 30])
```

C) 
```python
array([10, 20, 30, 1, 2, 3])
```

D) 
```python
array([ 9, 18, 27])
```

<!-- Answer: A) -->
