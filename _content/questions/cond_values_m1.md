Which of the following correctly represents the syntax of an `if` statement in Python?

A) 
```python
if condition:
    # code to execute
else
    # code to execute
```

B) 
```python
if (condition) {
    # code to execute
} else {
    # code to execute
}
```

C) 
```python
if condition:
    # code to execute
elif another_condition:
    # code to execute
else:
    # code to execute
```

D) 
```python
if condition then
    # code to execute
else if another_condition
    # code to execute
else
    # code to execute
```

<!-- Answer: C) -->
