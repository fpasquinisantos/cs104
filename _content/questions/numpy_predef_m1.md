Which code snippet correctly creates a NumPy array of zeros with shape `(3, 4)`?

A) 
```python
import numpy as np
array = np.zeros((3, 4))
```

B) 
```python
import numpy as np
array = np.ones((3, 4))
```

C) 
```python
import numpy as np
array = np.empty((3, 4))
```

D) 
```python
import numpy as np
array = np.zeros_array((3, 4))
```

<!-- Answer: A) -->
