Which code snippet correctly creates a NumPy array with a diagonal of ones and zeros elsewhere, with shape `(4, 4)`?

A) 
```python
import numpy as np
array = np.eye(4)
```

B) 
```python
import numpy as np
array = np.diag([1, 1, 1, 1])
```

C) 
```python
import numpy as np
array = np.identity(4)
```

D) 
```python
import numpy as np
array = np.zeros((4, 4))
```

<!-- Answer: A) -->
