How can you create a NumPy array from the tuple `(7.1, 8.2, 9.3)`?

A) 
```python
import numpy as np
array = np.array((7.1, 8.2, 9.3))
```

B) 
```python
import numpy as np
array = np.tuple((7.1, 8.2, 9.3))
```

C) 
```python
import numpy as np
array = np.fromtuple((7.1, 8.2, 9.3))
```

D) 
```python
import numpy as np
array = np.list((7.1, 8.2, 9.3))
```

<!-- Answer: A) -->
