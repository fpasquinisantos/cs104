- Which of the following is the correct way to write a multi-line comment in Python?

A) 
```python
/* This is a
   multi-line comment */
```

B) 
```python
# This is a
# multi-line comment
```

C) 
```python
""" This is a
    multi-line comment """
```

D) 
```python
-- This is a
-- multi-line comment
```

<!-- Answer: C) -->
