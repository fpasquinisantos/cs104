A) 
```python
if x = 10:
    # code to execute
```

B) 
```python
if x == 10:
    # code to execute
elif x != 10:
    # code to execute
```

C) 
```python
if (x == 10)
    # code to execute
```

D) 
```python

    # code to execute
else x < 10:
```
