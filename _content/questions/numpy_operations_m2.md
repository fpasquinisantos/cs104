Given the following NumPy arrays:

```python
import numpy as np
a = np.array([2, 4, 6])
b = np.array([1, 2, 3])
```

What will be the result of the following operation?

```python
result = a / b
```

A) 
```python
array([2.0, 2.0, 2.0])
```

B) 
```python
array([2, 4, 6])
```

C) 
```python
array([1.0, 2.0, 3.0])
```

D) 
```python
array([ 1.0,  2.0,  2.0])
```

<!-- Answer: A) -->
