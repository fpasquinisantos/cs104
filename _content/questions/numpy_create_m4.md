Which code snippet creates a NumPy array from a nested list `[[1, 2], [3, 4]]`?

A) 
```python
import numpy as np
array = np.array([[1, 2], [3, 4]])
```

B) 
```python
import numpy as np
array = np.arraylist([[1, 2], [3, 4]])
```

C) 
```python
import numpy as np
array = np.fromlist([[1, 2], [3, 4]])
```

D) 
```python
import numpy as np
array = np.create_array([[1, 2], [3, 4]])
```

<!-- Answer: A) -->
