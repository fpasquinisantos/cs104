Which code snippet correctly creates a NumPy array from a tuple `(4, 5, 6)`?

A) 
```python
import numpy as np
array = np.array((4, 5, 6))
```

B) 
```python
import numpy as np
array = np.tuple((4, 5, 6))
```

C) 
```python
import numpy as np
array = np.fromtuple((4, 5, 6))
```

D) 
```python
import numpy as np
array = np.create_array((4, 5, 6))
```

<!-- Answer: A) -->
