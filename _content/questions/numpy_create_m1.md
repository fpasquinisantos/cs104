How do you create a NumPy array from a list `[1, 2, 3]`?

A) 
```python
import numpy as np
array = np.array([1, 2, 3])
```

B) 
```python
import numpy as np
array = np.list([1, 2, 3])
```

C) 
```python
import numpy as np
array = np.fromlist([1, 2, 3])
```

D) 
```python
import numpy as np
array = np.create_array([1, 2, 3])
```

<!-- Answer: A) -->
