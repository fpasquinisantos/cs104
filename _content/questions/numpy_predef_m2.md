Which code snippet correctly creates a NumPy array with a range of numbers from 0 to 9?

A) 
```python
import numpy as np
array = np.arange(10)
```

B) 
```python
import numpy as np
array = np.range(10)
```

C) 
```python
import numpy as np
array = np.linspace(0, 9, 10)
```

D) 
```python
import numpy as np
array = np.seq(10)
```

<!-- Answer: A) -->
